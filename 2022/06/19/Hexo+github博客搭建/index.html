Hexo+github博客搭建

第1步: 安装Nodejs 和 Git

Nodejs版本: 16.14.0

​ ① 安装: Node.js安装及环境配置之Windows篇 - 刘奇云 - 博客园 (cnblogs.com)

​ ② 检测是否安装成功(cmd): node -v 和 npm -v

Git版本: 2.35.1.2

​ ① 安装: Git Windows安装教程（2.35.1.2版本）_跳舞的皮埃尔的博客-CSDN博客_git windows版

​ ② 检测是否安装成功(cmd): git --version

第2步: 安装Hexo

Hexo版本: 4.3.0

​ ① 安装: npm install hexo-cli -g

​ ② 检测是否安装成功(cmd): hexo -v

第3步: 搭建GitHub仓库

① gitHub注册(已注册忽略此步骤): Github注册+登录方法 - 哔哩哔哩 (bilibili.com)

② 创建仓库:
​

第4步: 通过SSH key将仓库绑定到本地

① 新建一个softwaremyBlog文件夹,然后在该文件夹中打开 Git Bash Here

​

②查看是否安装ssh: ssh

③生成ssh: ssh-keygen -t rsa -C "github注册时的邮箱地址"

④ 复制生成的ssh: 到路径C:\Users\用户名\下的.ssh文件夹中找到id_rsa.pub文件,用Notepad++打开并复制其中的内容

⑤ 将复制内容粘贴到GitHub上生成SSH密钥

​

​ ⑥ 检测是否绑定成功: ssh -T git@github.com

​ 

## 第5步: 本地开启博客

① 初始化hexo博客: hexo init

​

② 静态生成本地的hexo页面: hexo s

​

第6步: 将本地博客上传至互联网

① 修改配置文件(路径为‪D:\softwaremyBlog\_config.yml)

② 在softwaremyBlog文件夹中右击打开git bush, 安装hexo-deployer-git自动部署发布工具: npm install hexo-deployer-git --save

③输入邮箱和用户名:

​ git config --global user.email "你的邮箱"

​ git config --global user.name "你的gihub名字"

④ 清除之前生成文件: hexo clean

⑤ 生成页面: hexo g

⑥ 将本地内容上传到GitHub仓库: hexo d

第7步: 安装主题

① softwaremyBlog文件夹中右击打开git bush

输入: git clone https://github.com/zthxxx/hexo-theme-Wikitten.git themes/Wikitten安装主题

② 重写已覆盖站点文件夹中某些默认页面模板
$ cp -rf themes/Wikitten/_source/* source/
$ cp -rf themes/Wikitten/_scaffolds/* scaffolds/

③ 重命名以便于配置主题
cp -f themes/Wikitten/_config.yml.example themes/Wikitten/_config.yml

④ 修改_config.yml文件中的theme字段为wikitten(主题的名字)

⑤ 修改_config.yml文件中的其他配置文件


⑥ 修改 source\tags文件下的index内容

⑦ 更新到网络端
